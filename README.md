# franz-mueller.github.io

## Goal

This website serves as my portfolio. While I don't have a specific inclination toward learning web development, I find joy in acquiring some basic understanding of it.
Down the line, I intend to delve into script languages, but for now, I'm focused on solidifying my grasp of the fundamentals.

## Introduction

At this point, the website is fairly straightforward, comprising an index.html file, a styles.css file, and a resources folder housing images. It's hosted on GitHub Pages, and I rely on Visual Studio Code as my primary code editor. While I initially used Chat-GPT for the basics, the rest of the development involved extensive self-research and hands-on work.

## How I'm working on it

As the content expanded, I began implementing development branches.

### Testing

For testing I utilize a Docker container built on the nginx image for quicker visibility of changes compared to waiting for GitHub Pages to reflect them.
