The Goals


This website serves as my portfolio. While I don't have a specific inclination toward learning web development, I find joy in acquiring some basic understanding of it. Down the line, I intend to delve into script languages, but for now, I'm focused on solidifying my grasp of the fundamentals.

Introduction


At this point, the website is fairly straightforward, comprising an index.html file, a styles.css file, and a resources folder housing images. It's hosted on GitHub Pages, and I rely on Visual Studio Code as my primary code editor. While I initially used Chat-GPT for the basics, the rest of the development involved extensive self-research and hands-on work.

What I learned


I delved into Web development, learning about HTML and CSS. I actively applied new knowledge, experimenting with Docker containers, networks, and Git. Above all, this experience deepened my grasp on managing projects over extended durations. Making mistakes, especially in the beginning, has been a significant part of my learning journey. However, I've been evolving with every single one of those mistakes.

Testing


For testing I utilize a Docker container built on the nginx image for quicker visibility of changes compared to waiting for GitHub Pages to reflect them. In order to test the website on other devices I bind the container directly to my host (my PC). This way I can enter the website via the IP of my PC. In PowerShell it looks like this:

docker run -itd --network host --name Webmobiletester -v C:\_ProgrammingProjects\GitHubPage\Franz-Mueller.github.io:/usr/share/nginx/html nginx
