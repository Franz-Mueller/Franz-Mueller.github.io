Get data from Business Central table in Python via API

29.10.2023 | Python / AL | API / BC / ML
The goal is to write a Python script which can be executed by the user via Action-Button in Business Central. The Action Button runs a CodeUnit which then starts a Python script which pulls the data from the table behind the page, the action was on. The Script runs some Machien Learning Algorithmes on the Dataset and then gives the user some results. The first few steps are made: requesting Data via API and writing the first own ML-algorithm, even if it isn't overwhelmingly efficient.
